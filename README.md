# 4ws-Agricultural-Robot-Simulator
 A high-fidelity Scilab simulator for autonomous path tracking of a 4-wheel steering agricultural vehicle. Implements non-linear control strategies including Backstepping exact linearization, feedforward curvature anticipation, and Generalized Predictive Control (MPC-based). State estimation via Lyapunov-based adaptive observer. Validated on multiple trajectories with linear and Pacejka tire models.
